# Lib (Libraries & Config)

Конфигурация сторонних библиотек и инициализация синглтонов.

## Что здесь хранится?

1.  **Redux Store**: `store.ts`, `hooks.ts` — настройка хранилища.
2.  **i18n**: `i18n.ts` — конфигурация локализации.
3.  **Registry**: `registry.tsx` — для Styled Components или других провайдеров, требующих специальной настройки SSR.
4.  **Database Clients**: Инициализация Prisma, Supabase и т.д.

Это "инженерная" часть проекта, где мы настраиваем инструменты, которые потом используем в остальном приложении.
