# Components

Эта папка содержит переиспользуемые **UI-компоненты**, которые не зависят от бизнес-логики (или зависят минимально).

## Структура

Внутри `components` мы разделяем по типу:

*   **`ui/`**: Базовые примитивы.
    *   Кнопки (`Button`), Поля ввода (`Input`), Модалки (`Modal`), Карточки (`Card`).
    *   *Правило*: Они должны быть максимально абстрактными и принимать всё через пропсы.
*   **`layout/`**: Структурные блоки страницы.
    *   `Header`, `Footer`, `Sidebar`.
*   **`common/`**: Общие, более сложные компоненты, используемые в разных частях приложения.
    *   Например, `SEOHead`, `LoadingSpinner`.

## Правила

1.  Каждый компонент должен находиться в своей папке (например, `components/ui/Button/Button.tsx`).
2.  Используйте `index.ts` для экспорта.
3.  Стили (Styled Components) могут быть в том же файле или рядом (`Button.styles.ts`).
