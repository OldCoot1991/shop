# Layout Components

Компоненты, определяющие каркас страниц.

## Примеры
- `Header`: Верхняя навигация, логотип, корзина.
- `Footer`: Подвал сайта.
- `Sidebar`: Боковое меню (админка или категории).
- `Container`: Обертка для контента с максимальной шириной.

Эти компоненты могут подключаться к Redux (например, отображать количество товаров в корзине в хедере).
