# Hooks

Глобальные пользовательские React хуки.

## Правила

Здесь должны лежать только **общие** хуки, которые могут быть полезны в любом месте приложения.
Хуки, специфичные для конкретной фичи (например, `useCartTotal`), лучше держать в `features/cart/hooks`.

## Примеры
- `useDebounce.ts`: Отложенный вызов функции (для поиска).
- `useClickOutside.ts`: Закрытие модалки по клику вне области.
- `useMediaQuery.ts`: Адаптивность в JS.
- `useLocalStorage.ts`: Удобная работа с локальным хранилищем.
