# Utils (Utilities)

Папка для чистых вспомогательных функций (Pure Functions).

## Характеристики
- **Без состояния**: Функции не должны хранить данные.
- **Без React-логики**: Здесь **не** должно быть хуков или компонентов. Только чистый JS/TS.
- **Тестируемость**: Логику отсюда легко покрыть Unit-тестами.

## Примеры
- `formatDate.ts`: Форматирование даты.
- `currency.ts`: Форматирование цен (например, добавление знака валюты).
- `validation.ts`: Функции валидации email, паролей.
- `constants.ts`: Глобальные константы (названия сайтов, лимиты).
