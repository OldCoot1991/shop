# Styles

Глобальные стили, темы и дизайн-токены.

## Содержимое

*   **`theme.ts`**: Определение тем (Light / Dark) и переменных (цвета, шрифты, отступы).
*   **GlobalStyles**: Глобальный сброс CSS (Reset), базовые стили для `body`, `html`.
*   **Mixins / Variables**: Если используются, вспомогательные функции для CSS.

Локальные стили компонентов должны находиться рядом с самими компонентами, а не здесь.
