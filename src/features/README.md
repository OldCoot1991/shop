# Features (Business Logic)

Эта папка содержит **функциональные модули** приложения. Это "вертикальные слайсы" (Vertical Slices), сгруппированные по бизнес-сущностям.

## Структура

Пример структуры папок внутри `features`:

*   **`auth/`**: Все, что связано с авторизацией.
    *   `LoginForm`, `RegisterForm`.
    *   `authSlice.ts` (Redux).
    *   `useAuth.ts` (Hook).
*   **`cart/`**: Корзина покупок.
    *   `CartList`, `CartItem`.
    *   `cartSlice.ts`.
*   **`products/`**: Каталог товаров.
    *   `ProductList`, `ProductCard`.
    *   `productsApi.ts` (RTK Query или API calls).

## Философия

Мы стараемся держать всю логику конкретной фичи **внутри** её папки. Это делает код модульным. Если вы удалите папку `cart`, у вас исчезнет только корзина, а не сломается полсайта.
