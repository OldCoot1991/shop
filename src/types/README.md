# Types

Глобальные определения типов TypeScript (`.ts` или `.d.ts`).

## Что здесь хранится?

1.  **Сущности БД**: Типы, описывающие данные с бэкенда.
    *   `User.ts`
    *   `Product.ts`
    *   `Order.ts`
2.  **API Responses**: Типы ответов сервера (Pagination, ErrorResponse).
3.  **Enums**: Глобальные перечисления (например, `UserRole`, `OrderStatus`).

*Примечание*: Props для конкретных компонентов лучше держать прямо в файле компонента или рядом с ним.
